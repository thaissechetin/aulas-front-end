Trabalhando com listas no HTML

Listas são estruturas usadas quando precisamos representar conteúdo na forma de uma relação de ítens.

Lista não-ordenada

  • Mouse
  • Teclado
  • Monitor
  • Headset

Lista ordenada

  1. Ligar o computador
  2. Entrar com o usuário/senha
  3. Aguardar o sistema carregar
  4. Procurar pelo programa desejado
  5. Abrir o programa

Para abrir a lista ordenada a partir de um número que não seja o 1, usa-se o " ol start = e o numero que deseja começar :

  1. Ligar o computador
  2. Digitar usuário e a senha
  3. Aguardar o sistema carregar
